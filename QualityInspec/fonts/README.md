# 폰트 파일 설치

PDF 생성을 위해 한글 폰트가 필요합니다.

## 설치 방법

1. NanumGothic.ttf 또는 다른 한글 폰트 파일을 이 폴더에 복사
2. Windows의 경우: `C:\Windows\Fonts\malgun.ttf`를 복사 가능
3. 또는 무료 폰트 다운로드:
   - Noto Sans KR: https://fonts.google.com/noto/specimen/Noto+Sans+KR
   - Nanum Gothic: https://hangeul.naver.com/font

## 파일명

`NanumGothic.ttf` 또는 `malgun.ttf`로 저장하세요.
